1 デザインコンセプト

デザインキーワード
デザインの方向性を表すキーワード3つとその説明

1. 和モダン

浅草らしい日本文化の雰囲気を伝えながら、現代的で洗練された印象にすることで、若い旅行者にも親しみやすくします。

生成り・墨色・えんじなど和を感じる色を使い、明朝体や和柄モチーフを控えめに取り入れて表現します。

2. 上質感

香りづくり体験を、旅先で味わう特別で価値のある時間として伝えるため、落ち着きと高級感のあるデザインを目指します。

余白を活かしたレイアウトや、素材感・手仕事感が伝わる写真を使い、上品で丁寧な印象に仕上げます。

3. ときめき

「体験してみたい」「思い出として残したい」と感じてもらうために、感性に響く魅力を大切にします。

やわらかな雰囲気の写真や繊細な装飾、印象に残るコピーを用いて、心が動く世界観を表現します。

4 UIコンポーネント

ボタン
プライマリ・セカンダリ・サーフェスの3種類のボタンスタイル（実物のHTMLサンプル付き）

ボタンは Noto Sans JP・20px・700・角丸（--radius-md）を基本とし、プライマリ（.btn--primary）／セカンダリ（.btn--secondary）／サーフェス（.btn--surface）で役割を区別する。ホバー時は背景色・枠線色を調整して操作性を高める。

予約する 詳細を見る 内容を確認

<div class="button-group">
  <a class="btn btn--primary" href="#">予約する</a>
  <a class="btn btn--secondary" href="#">詳細を見る</a>
  <a class="btn btn--surface" href="#">内容を確認</a>
</div>

.button-group {
  display: flex;

フォーム要素
テキスト入力・セレクト・テキストエリア・チェックボックス・ラジオボタン（実物のHTMLサンプル付き）

フォーム要素は .field / .input / .select / .textarea を基本とし、フォーカス時はメインカラーで強調、エラー時は .is-error と .field-error で状態を明確にする。チェックボックスとラジオボタンは .choice を使用し、accent-color をブランドカラーに合わせて統一する。

5 レイアウトルール

レイアウト基本ルール
グリッド・余白・角丸・シャドウ等の数値ルール

コンテンツ最大幅は1200px（--container）とし、左右余白は calc(100% - 32px) を基本とする。レイアウトは各グリッドの gap: 20px を基本とし、@media (min-width: 768px) で主要レイアウトを2カラム化する。セクション間の余白はスマートフォン72px（--section-space）、タブレット96px、PC120pxとする。角丸はカード等で12px（--radius-lg）、ボタンや入力欄で8px（--radius-md）、小要素で4px（--radius-sm）を基準とする。シャドウは 0 8px 24px rgba(47, 42, 40, 0.06)（--shadow-sm）を基本とし、強調は 0 12px 32px rgba(47, 42, 40, 0.10)（--shadow-md）を使用する。ブレークポイントはPC 1024px以上、タブレット 768px〜1023px、スマートフォン 767px以下とする。
