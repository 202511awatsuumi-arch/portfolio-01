1 テスト対象の範囲

テスト対象の範囲
対象機能とテスト種別、テスト環境、テスト期間を記載してください。

【テスト対象の範囲】
本テストでは、「香り結び」の管理システムおよび公開お問い合わせ機能を対象とする。
対象機能は以下のとおり。
・F-001 ログイン
・F-002 ログアウト
・F-003 ユーザー一覧表示
・F-004 ユーザー新規登録
・F-005 ユーザー編集
・F-006 ユーザー削除
・F-007 お問い合わせ一覧表示
・F-008 削除済みお問い合わせ一覧表示
・F-009 お問い合わせ詳細表示
・F-010 お問い合わせ編集
・F-011 お問い合わせ削除
・F-012 お問い合わせ復元
・F-013 お問い合わせ送信
・F-014 入力バリデーション
・F-015 ページネーション

【テスト種別】
・機能テスト：各機能が仕様どおりに動作することを確認する
・画面テスト：画面表示、入力欄、ボタン、一覧表示、画面遷移が正しいことを確認する
・セキュリティテスト：ログイン認証、未認証アクセス制御、入力値のバリデーション、CSRF対策の基本動作を確認する

【テスト環境】
・ブラウザ：Google Chrome、Microsoft Edge、Firefox
・OS：Windows、macOS、iOS、Android

【テスト期間】
・2026-04-22 ～ 2026-04-24
